Algorítimos

Instruções em sequência lógica para determinada ação, próximo tópico.

Brincadeiras a parte, aí vai uma algorítimo para somar dois números primos:

Escolha um número
É um número primo?
Se sim, anotar no papel
Se não, escolher outro
Escolha outro número primo
Execute a soma dos números
Escreva o resultado no papel

Agora isso na linguagem Java:

System.out.print("Escolha um número primo");

int numeroUm = entradaDeDados.nextInt();

validacaoDoNumero();

System.out.print("Escolha outro número primo");

validacaoDoNumero();

int numeroDois = entradaDeDados.nextInt();

System.out.print("Resultado da Soma: " numeroUm + NumeroDois);

Se você apenas usar os conhecimentos que adquiriu por toda sua vida, é possível entender o que eu quis fazer usando a linguagem Java. Ou seja, um Algorítimo é basicamente uma instrução, e para ser executada o individuo ou maquina responsável pela execução deve entender, e para isso utilizamos comandos, codificados na forma de idiomas para seres vivos, ou de linguagem de programação para máquinas.

Lógica de Programação

É a lógica que rege como estruturar seu algorítimo. Para escrever o algorítimo da soma de dois números primos utilizei a sintaxe da linguagem Java, a sintaxe é a responsável por especificar como deve ser usado uma linguagem.

Utilizei de uma estrutura que me permite apresentar uma informação na tela, dentro do parenteses coloquei a informação.

System.out.print("Escolha um número primo");

Depois declarei uma variável do conjunto dos números inteiros cujo nome foi numeroUm, esse nome me permite apelidar um espaço na memória da minha máquina para quando eu precisar usar eu achar esse espaço pelo nome, em seguida apelidei uma função que informa para a máquina que irá entrar um dado, e esse dado será armazenado no espaço da memória chamado numeroUm. Repeti a mesma lógica para o outro número.

int numeroUm = entradaDeDados.nextInt();

Em seguida validei através de uma função se o número era primitivo ou não. Uma função basicamente faz uma pergunta e recebe uma resposta, e depois retorna um resultado ou não.

validacaoDoNumero();

No final do código apresentei o resultado concatenando uma frase com a soma dos números.

System.out.print("Resultado da Soma: " numeroUm + NumeroDois);
